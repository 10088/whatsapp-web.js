Class Client

Starting point for interacting with the WhatsApp Web API

param options

Client options

param options.authStrategy

Determines how to save and restore sessions. Will use LegacySessionAuth if options.session is set. Otherwise, NoAuth will be used.

param options.authTimeoutMs

Timeout for authentication selector in puppeteer

param options.puppeteer

Puppeteer launch options. View docs here: https://github.com/puppeteer/puppeteer/

param options.qrMaxRetries

How many times should the qrcode be refreshed before giving up

param options.restartOnAuthFail

This option should be set directly on the LegacySessionAuth.

param options.session

Only here for backwards-compatibility. You should move to using LocalAuth, or set the authStrategy to LegacySessionAuth explicitly.

param options.takeoverOnConflict

If another whatsapp web session is detected (another browser), take over the session in the current browser

param options.takeoverTimeoutMs

How much time to wait before taking over the session

param options.userAgent

User agent to use in puppeteer

param options.ffmpegPath

Ffmpeg path to use when formating videos to webp while sending stickers

param options.bypassCSP

Sets bypassing of page's Content-Security-Policy.
